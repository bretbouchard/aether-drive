## Drum Machine - Drum Synthesis (8 Songs)

**Instrument Overview**:
- 15-voice drum synthesis (Kick, Snare, Hi-Hats, Clap, Toms, Cymbals, Cowbell, Shaker, Tambourine, Percussion, Special)
- 16-track pattern sequencer (16 steps per track)
- Drill mode for IDM/breakbeat generation
- Per-step probability, flam, roll, and burst
- Timing roles: Pocket/Push/Pull

### Demo Songs:

**1. "Classic 808 Hip Hop"** (Basic Pattern)
- **Feature**: 15-voice drum synthesis
- **Preset**: Classic 808 Kit
- **Obscure Feature**: Per-step probability for human feel
- **Technique**: Four-on-the-floor kick, snare on 2/4, hi-hat on offbeats

**2. "Aphex Snare Hell"** (Drill Mode Showcase)
- **Feature**: Drill mode with bursts
- **Preset**: Aphex Drill Kit
- **Obscure Feature**: Automation lane for compositional drill sequencing
- **Technique**: 13 drill parameters create chaotic snare rolls and micro-bursts

**3. "Timing Roles Groove"** (Pocket/Push/Pull)
- **Feature**: Timing roles per track
- **Preset**: Groove Kit
- **Obscure Feature**: Push/Pull creates aggressive or laid-back feel
- **Technique**: Kick = Pocket (steady), Snare = Push (aggressive), Hi-hat = Pull (laid back)

**4. "Probability Funk"** (Per-Step Probability)
- **Feature**: Probability per step
- **Preset**: Funk Kit
- **Obscure Feature**: Probability < 1.0 creates variation
- **Technique**: Set hi-hat probability to 0.7-0.9 for ghost notes

**5. "Velocity Accents"** (Dynamic Pattern)
- **Feature**: Velocity accents
- **Preset**: Dynamic Kit
- **Obscure Feature**: Velocity layering for realism
- **Technique**: Accented kicks (120) vs ghost kicks (90)

**6. "Swing Time"** (Groove Control)
- **Feature**: Swing per pattern
- **Preset**: Swing Kit
- **Obscure Feature**: Swing affects timing grid
- **Technique**: 57% swing for classic hip-hop feel

**7. "Fill Policy"** (Automatic Fills)
- **Feature**: Fill policy system
- **Preset**: Fill Kit
- **Obscure Feature**: Auto-generates fills with decay
- **Technique**: 2-step fills with 0.15 decay per step

**8. "Gate Chaos"** (Gate Policy)
- **Feature**: Gate policy for tension
- **Preset**: Gate Kit
- **Obscure Feature**: Silence → Burst explosions
- **Technique**: 25% silence chance, 60% burst chance

## LOCAL_GAL - Acid Synthesizer (10 Songs)

**Instrument Overview**:
- Unique "Feel Vector" synthesis (rubber, bite, hollow, growl)
- Built-in pattern sequencer with probability
- Parameter morphing system
- 16-voice polyphonic synthesis

### Demo Songs:

**1. "Acid Burns"** (Classic 303 Showcase)
- **Feature**: Feel vector "bite" for aggressive resonance
- **Preset**: Custom Acid Techno preset
- **Obscure Feature**: Pattern probability for generative acid lines
- **Technique**: Use high bite (0.8-0.9) for screaming resonance, low rubber (0.1) for snappy envelopes

**2. "Glide Like Rubber"** (Portamento Showcase)
- **Feature**: Feel vector "rubber" for glide timing
- **Preset**: Rubber Bass Init
- **Obscure Feature**: Rubber value controls both glide time and envelope timing
- **Technique**: High rubber (0.7-0.9) creates smooth pitch slides between notes

**3. "Hollow Ambience"** (Filter Exploration)
- **Feature**: Feel vector "hollow" for warmth
- **Preset**: Warm Pad Init
- **Obscure Feature**: Hollow affects filter cutoff differently per voice
- **Technique**: High hollow (0.8-0.9) opens filter for warm, mellow pads

**4. "Growl Bass"** (Saturation Showcase)
- **Feature**: Feel vector "growl" for drive
- **Preset**: Aggressive Bass
- **Obscure Feature**: Growl adds waveshaping distortion before filter
- **Technique**: High growl (0.7+) with high bite creates distorted acid bass

**5. "Morphing Dreams"** (Parameter Morphing)
- **Feature**: Real-time parameter morphing
- **Preset**: Morph Pad A & B
- **Obscure Feature**: Morph position interpolates all feel vector parameters
- **Technique**: Automate morph position to transform between presets

**6. "Probability Groove"** (Pattern Sequencer)
- **Feature**: Pattern probability for variation
- **Preset**: Random Acid
- **Obscure Feature**: Probability creates non-repeating patterns
- **Technique**: Set velocity probability to 0.5-0.7 for controlled chaos

**7. "Swing Time"** (Pattern Swing)
- **Feature**: Pattern swing for groove
- **Preset**: Swing Acid
- **Obscure Feature**: Swing affects timing of pattern steps
- **Technique**: 57% swing creates classic shuffle feel

**8. "Polyphonic Bliss"** (16-Voice Showcase)
- **Feature**: Maximum polyphony
- **Preset**: Bright Pad
- **Obscure Feature**: Voice stealing algorithm prioritizes oldest notes
- **Technique**: Layer pads with long release to test voice management

**9. "Acid Arpeggio"** (Pattern + Feel)
- **Feature**: Pattern sequencer + feel vector
- **Preset**: Arpeggio Acid
- **Obscure Feature**: Pattern length independent of bar length
- **Technique**: 16-step pattern with feel vector modulation

**10. "Feel The Morph"** (Complete Showcase)
- **Feature**: All feel vectors + morphing
- **Preset**: Custom multi-preset
- **Obscure Feature**: Morph interpolates between 4 feel vector states
- **Technique**: Start soft (hollow), morph to aggressive (bite + growl)
